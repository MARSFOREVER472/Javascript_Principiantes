Beginner Exercises: Part 1

Exercise 1:

Write a Javascript Program to check 2 numbers and return "true" if one of the numbers is 100 or if the sum of the 2 numbers is 100.

Solution:

const isEqualTo100 = (a, b) => a === 100 || b === 100;

console.log(isEqualTo100(100,0)); // PRINT THE SUM OF THIS 2 NUMBERS EQUALS TO 100 AND RETURNS TRUE (a,b).

console.log(isEqualTo100(0,100));
